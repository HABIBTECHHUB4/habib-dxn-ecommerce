# Habib DXN E-commerce - সেটআপ গাইড

## প্রয়োজনীয় সফটওয়্যার

আপনার কম্পিউটারে নিম্নলিখিত ইনস্টল করা থাকতে হবে:

- **Node.js** (v16 বা তার উপরে) - [nodejs.org](https://nodejs.org) থেকে ডাউনলোড করুন
- **pnpm** (প্যাকেজ ম্যানেজার) - অথবা npm ব্যবহার করতে পারেন

## ইনস্টলেশন ধাপ

### ১. প্রজেক্ট ফোল্ডার খুলুন
```bash
cd habib-dxn-ecommerce
```

### ২. ডিপেন্ডেন্সি ইনস্টল করুন
```bash
pnpm install
```

অথবা npm ব্যবহার করলে:
```bash
npm install
```

### ৩. ডেভেলপমেন্ট সার্ভার চালু করুন
```bash
pnpm run dev
```

অথবা npm ব্যবহার করলে:
```bash
npm run dev
```

### ৪. ব্রাউজারে খুলুন
ডেভেলপমেন্ট সার্ভার চালু হলে, ব্রাউজারে এই ঠিকানা খুলুন:
```
http://localhost:3000
```

## প্রোডাকশনের জন্য বিল্ড করুন

ওয়েবসাইট প্রোডাকশনের জন্য প্রস্তুত করতে:

```bash
pnpm run build
```

এটি একটি `dist` ফোল্ডার তৈরি করবে যা আপনি যেকোনো হোস্টিং সেবায় আপলোড করতে পারেন।

## প্রোডাকশন সার্ভার চালু করুন

```bash
pnpm run start
```

### পণ্য যোগ/সংশোধন করুন
`client/src/lib/products.ts` ফাইল খুলুন এবং পণ্য যোগ করুন:

```typescript
{
  id: "9",
  name: "নতুন পণ্যের নাম",
  price: 1000,
  description: "পণ্যের বর্ণনা",
  image: "ছবির URL",
  category: "ক্যাটাগরি",
  benefits: ["সুবিধা ১", "সুবিধা ২"],
}
```

### পেমেন্ট পরিমাণ পরিবর্তন করুন
`client/src/pages/Checkout.tsx` ফাইলে খুঁজুন:
```typescript
const codAmount = 130; // এই সংখ্যা পরিবর্তন করুন
```

### ডিজাইন পরিবর্তন করুন
- রঙ পরিবর্তন: `client/src/index.css` এ CSS ভেরিয়েবল সংশোধন করুন
- লেআউট পরিবর্তন: প্রতিটি পেজ ফাইলে Tailwind ক্লাস সংশোধন করুন

## সাধারণ সমস্যা ও সমাধান

### সমস্যা: `pnpm: command not found`
**সমাধান:** npm ব্যবহার করুন বা pnpm ইনস্টল করুন:
```bash
npm install -g pnpm
```

### সমস্যা: পোর্ট 3000 ইতিমধ্যে ব্যবহৃত
**সমাধান:** অন্য পোর্ট ব্যবহার করুন:
```bash
pnpm run dev -- --port 3001
```

### সমস্যা: মডিউল খুঁজে পাওয়া যাচ্ছে না
**সমাধান:** ডিপেন্ডেন্সি পুনরায় ইনস্টল করুন:
```bash
rm -rf node_modules pnpm-lock.yaml
pnpm install
```
