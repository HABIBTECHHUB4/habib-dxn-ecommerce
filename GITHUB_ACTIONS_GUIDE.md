# GitHub Actions অটোমেটিক সিঙ্ক গাইড

## কি হচ্ছে?

আমরা GitHub Actions সেটআপ করেছি যাতে:
- **Manus-এ যখন এডিট করবেন** → স্বয়ংক্রিয়ভাবে **GitHub এ আপডেট হয়ে যাবে**
- কোনো ম্যানুয়াল কমান্ড চালাতে হবে না
- সবসময় GitHub এবং Manus সিঙ্ক থাকবে

## কিভাবে কাজ করে?

### ধাপ ১: Manus-এ এডিট করুন
- আপনার প্রথম Manus অ্যাকাউন্টে লগইন করুন
- ওয়েবসাইট এডিট করুন (পণ্য যোগ করুন, ডিজাইন পরিবর্তন করুন, ইত্যাদি)
- Manus-এ **Checkpoint সেভ করুন** (গুরুত্বপূর্ণ!)

### ধাপ ২: স্বয়ংক্রিয় সিঙ্ক
- GitHub Actions স্বয়ংক্রিয়ভাবে আপনার পরিবর্তন GitHub এ পুশ করবে
- এটি কয়েক সেকেন্ড থেকে কয়েক মিনিট সময় নিতে পারে

### ধাপ ৩: দ্বিতীয় অ্যাকাউন্টে আপডেট পান
- আপনার দ্বিতীয় Manus অ্যাকাউন্টে GitHub থেকে পুল করুন
- অথবা GitHub থেকে সরাসরি ডাউনলোড করুন

## স্বয়ংক্রিয় সিঙ্ক চেক করুন

### GitHub Actions Status দেখুন
1. GitHub এ যান: https://github.com/HABIBTECHHUB4/habib-dxn-ecommerce
2. **Actions** ট্যাব ক্লিক করুন
3. সবুজ চেকমার্ক মানে সফল সিঙ্ক
4. লাল X মানে কোনো সমস্যা হয়েছে

## কখন সিঙ্ক হয়?

GitHub Actions নিম্নলিখিত পরিস্থিতিতে চলে:

### ১. স্বয়ংক্রিয় (Automatic)
- Manus থেকে কোনো পুশ হলে
- GitHub এ কোনো পরিবর্তন হলে

### ২. ম্যানুয়াল (Manual)
- GitHub এ **Actions** ট্যাব থেকে ম্যানুয়ালি চালু করতে পারেন

## সিঙ্ক সমস্যা হলে

### সমস্যা ১: সিঙ্ক হচ্ছে না
**সমাধান:**
1. GitHub এ **Actions** ট্যাব চেক করুন
2. লাল X দেখলে ক্লিক করে বিস্তারিত দেখুন
3. সাধারণত এটি টোকেন সমস্যা হয়

### সমস্যা ২: কনফ্লিক্ট হচ্ছে
**সমাধান:**
1. GitHub এ সর্বশেষ কোড পুল করুন
2. Manus-এ সেই কোড আপডেট করুন
3. তারপর আবার এডিট করুন

## দ্বিতীয় অ্যাকাউন্টে কিভাবে আপডেট পাবেন?

### অপশন ১: GitHub থেকে পুল করুন
```bash
cd habib-dxn-ecommerce
git pull origin main
```

### অপশন ২: Manus-এ GitHub থেকে রিইম্পোর্ট করুন
1. দ্বিতীয় Manus অ্যাকাউন্টে লগইন করুন
2. প্রজেক্ট সেটিংসে যান
3. **Sync from GitHub** অপশন খুঁজুন
4. সিঙ্ক করুন

## গুরুত্বপূর্ণ নোট

⚠️ **Manus-এ Checkpoint সেভ করা আবশ্যক**
- Checkpoint সেভ না করলে সিঙ্ক হবে না
- প্রতিটি গুরুত্বপূর্ণ পরিবর্তনের পর Checkpoint সেভ করুন

⚠️ **টোকেন নিরাপত্তা**
- আপনার GitHub টোকেন সুরক্ষিত রাখুন
- কাউকে শেয়ার করবেন না

⚠️ **দুটি জায়গায় একসাথে এডিট করবেন না**
- Manus এবং GitHub এ একসাথে এডিট করলে কনফ্লিক্ট হতে পারে
- একটি জায়গায় এডিট করুন, সিঙ্ক হওয়ার পর অন্যটায় করুন

## সিঙ্ক ডিসেবল করতে চাইলে

যদি স্বয়ংক্রিয় সিঙ্ক বন্ধ করতে চান:

1. GitHub এ যান
2. **Settings** → **Actions** → **General**
3. **Actions permissions** → **Disable all**

## সাহায্য প্রয়োজন?

যদি কোনো সমস্যা হয়:
1. GitHub Actions লগ চেক করুন
2. সর্বশেষ কোড পুল করুন
3. Manus-এ Checkpoint সেভ করুন
